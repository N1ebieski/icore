Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

https://icore.test

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_ACCESS_TOKEN}" or by sending cookie session credentials with Referer/Origin header from stateful domain.

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting tokens tab in your profile and clicking Create API token or sending authentication request via API.

Show user

requires authentication

Show authenticated User resource

Available only to users with permission: api.access.
Example request:
curl --request GET \
    --get "https://icore.test/api/user" \
    --header "Authorization: Bearer {YOUR_ACCESS_TOKEN}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://icore.test/api/user"
);

const headers = {
    "Authorization": "Bearer {YOUR_ACCESS_TOKEN}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'https://icore.test/api/user',
    [
        'headers' => [
            'Authorization' => 'Bearer {YOUR_ACCESS_TOKEN}',
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Token

Create access token and (optional) refresh token.

Access token expires after 2 hours. Refresh token expires after 1 year.
Example request:
curl --request POST \
    "https://icore.test/api/token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"kontakt@demo.icore.intelekt.net.pl\",
    \"password\": \"demo1234\",
    \"remember\": true
}"

const url = new URL(
    "https://icore.test/api/token"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "kontakt@demo.icore.intelekt.net.pl",
    "password": "demo1234",
    "remember": true
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'https://icore.test/api/token',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'email' => 'kontakt@demo.icore.intelekt.net.pl',
            'password' => 'demo1234',
            'remember' => true,
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
